# 前端 UI 重构测试用例

## 自动验证

在项目根目录执行：

```bash
cd frontend
npm run typecheck
npm run build
cd ..
python -m pytest tests
git diff --check
```

## 启动方式

本地 API：

```bash
uvicorn app.api.server:app --reload --host 127.0.0.1 --port 8000
```

前端开发服务器：

```bash
cd frontend
npm run dev -- --host 127.0.0.1
```

默认访问地址：

```text
http://127.0.0.1:1420/
```
